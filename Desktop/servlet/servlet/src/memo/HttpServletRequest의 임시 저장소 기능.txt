HttpServletRequest의 임시 저장소 기능
→ 나는 처음에 이 기능을 보고, 어차피 모든 요청 데이터는 HttpServletRequest에 이미 담겨져 있는데 왜 굳이 임시 저장소가 필요할까?라는 생각을 했는데
용도가 아예 다르다. 
임시 저장소 기능은 요청 데이터에 없는, 예를 들어 서버에서 만든 데이터를 저장해서 jsp라던지 뷰 템플릿에 데이터를 전달하고 싶을 떄 사용하는 것이다. 
 
ex)
HttpServletRequest.setAttribute(serverData , "Hello") // 서버 데이터를 임시 저장소에 저장
request.getRequestDispatcher("/view.jsp").forward(request, response); // jsp에 전달
